# 🌍 Travel and Tourism Integrated with Agentic AI ✈️🤖

## 🔥 Overview

This project is an intelligent **Travel and Tourism Recommendation System** powered by **Agentic AI**. It helps users plan their trips efficiently by analyzing their preferences and automatically suggesting the best destinations, travel options, and accommodations.

The system acts like a **smart travel assistant 🧠**, making decisions and recommendations dynamically based on user input, weather conditions, and available services.

---

## ⚡ Features

* 🧳 Personalized travel destination recommendations
* 🤖 Agentic AI-based decision making
* 🌦️ Weather-aware travel suggestions
* 🗺️ Integration with maps for navigation
* 🏨 Hotel and accommodation recommendations
* 🚆✈️ Travel booking suggestions (train & flight)
* 📊 Smart scoring system for selecting best destinations
* 💻 User-friendly web interface

---

## 📌 Requirements

Make sure you have Python installed on your system. Then install all dependencies:

```bash
pip install -r requirements.txt
```

---

## 🚀 Setup and Running the Project

1. **Clone the Repository** 📥

   ```bash
   git clone https://github.com/your-username/travel-tourism-agentic-ai.git
   cd travel-tourism-agentic-ai
   ```

2. **Install Dependencies** ⚙️

   ```bash
   pip install -r requirements.txt
   ```

3. **Setup API Keys 🔑**

   * Add your API keys for:

     * Weather API 🌦️
     * Google Maps API 🗺️
   * Store them in a `.env` file or directly in the config file.

4. **Run the Application 🎯**

   ```bash
   python app.py
   ```

5. **Open in Browser 🌐**

   * Go to: `http://localhost:5000`

---

## 🛠️ How It Works

1. The user provides inputs such as budget, preferences, and travel type.
2. The system evaluates multiple destinations using a **scoring algorithm**.
3. Agentic AI selects the **top 2–3 best locations**.
4. The user confirms a destination.
5. The system suggests:

   * 🏨 Hotels
   * 🚆✈️ Travel options
   * 🗺️ Route and itinerary
6. Final recommendations are displayed for booking and planning.

---

## 🧠 What is Agentic AI?

Agentic AI refers to systems that can **autonomously make decisions** and take actions based on user goals.
In this project, it:

* Analyzes user preferences
* Evaluates multiple options
* Selects optimal travel plans automatically
